# Code of Conduct

## Our Pledge

CyberMind is an offensive security tool built for authorized security research, penetration testing with written permission, CTF competitions, and cybersecurity education.

We are committed to making participation in this project a respectful and productive experience for everyone.

## Standards

**Expected behavior:**
- Use respectful and professional language
- Focus on constructive feedback and improvements
- Accept differing viewpoints and experiences
- Show empathy toward other contributors

**Unacceptable behavior:**
- Using the project or its tools against unauthorized targets
- Harassment, trolling, or personal attacks
- Publishing others' private information without consent
- Any conduct that violates applicable laws

## Authorized Use Only

This project is strictly for:
- Authorized penetration testing (written permission required)
- Bug bounty programs within scope
- CTF competitions
- Security research and education

**Never use CyberMind against systems you do not own or have explicit written authorization to test.**

## Enforcement

Violations may result in removal from the project. Report issues to the maintainer at github.com/thecnical.
